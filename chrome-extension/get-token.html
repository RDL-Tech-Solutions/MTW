🔑 Obter Token de Autenticação

Para usar a extensão Chrome

⚠️ Importante: Você precisa ser um administrador da plataforma para usar a extensão.

📋 Solução para Erro de CORS:

⚠️ Problema: Arquivos HTML locais não podem fazer requisições HTTP devido ao CORS.

✅ Solução 1: Via Painel Admin (Mais Fácil)

  1. Acesse o painel admin: http://localhost:3001
  2. Faça login com sua conta de administrador
  3. Pressione F12 para abrir o DevTools
  4. Vá para a aba Console
  5. Digite: localStorage.getItem('token')
  6. Copie o token (sem as aspas)

✅ Solução 2: Via Script Node.js

  1. Abra o terminal na pasta chrome-extension/
  2. Execute: node get-token.js
  3. Siga as instruções e copie o token

✅ Solução 3: Via cURL

curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"seu-email@exemplo.com","password":"sua-senha"}'
